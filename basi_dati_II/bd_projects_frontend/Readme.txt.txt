# Air Quality Insights: Explore Pollution Levels in Real-Time

L'inquinamento atmosferico è una seria minaccia per la salute umana e l'ambiente a causa delle emissioni inquinanti da attività umane, con effetti negativi sui polmoni, il sistema cardiovascolare e il clima globale. La raccolta di dati sull'inquinamento atmosferico è essenziale per monitorare l'impatto e adottare misure efficaci per ridurlo.

## Struttura del progetto

Il progetto è articolato in due cartelle principali:

1. **Cartella principale**: Contiene i file relativi al backend del progetto.

   - AirQuality.py: File che definisce la classe AirQuality.
   - BDConnectionPool.py: File che gestisce la connessione al database.
   - CleanDataset.py: File che si occupa della pulizia del dataset.
   - Pages.py: File che gestisce le pagine dell'applicazione.
   - QueryList.py: File che contiene l'elenco delle query per il database.

2. **bd_projects_frontend**: Contiene i file relativi al frontend del progetto.

   - index.html: Pagina principale dell'applicazione.
   - test_query.html: Contiene gli elementi per effettuare le interrogazioni.
   - style.css: File CSS per lo stile dell'applicazione.
   - script.js: File JavaScript per le funzionalità interattive dell'applicazione.

Per quanto riguarda l'avvio, il main è presente nel file Pages.py.
